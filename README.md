🎮 클론 인스타그램 기능  
-------------  

- 로그인 및 회원가입 기능
- 메인화면에서 다른 사람들이 올린 글 확인 가능
- 글쓰기 기능을 통해 내가 찍은 사진을 공유
- 댓글 기능을 통해 다양한 사람들과 소통
- 프로필 페이지에서 내 정보를 확인
- 좋아요 기능으로 다른 사람 글 추천
